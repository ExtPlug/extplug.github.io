ExtPlug

ExtPlug is an advanced, flexible extension for plug.dj.

Installation

Bookmark

The easiest way to use ExtPlug is to install the below bookmarklet by dragging it to your bookmarks bar. Then, simply press the bookmark on a plug.dj page to start using ExtPlug!

Bookmarklet

Userscript

If you want to auto-load ExtPlug every time you open up plug.dj, without one of the ExtPlug browser extensions, you can use the userscript version instead. Install the Greasemonkey addon for Firefox, or the Tampermonkey extension for Chrome, and press the button below!

Features

Plugins

All ExtPlug functionality comes in the form of plugins. This means that you can easily mix and match functionality that you like, and leave out the bits that you don't like.

Bundled Plugins

Bundled plugins are installed by default. You can still remove them, if you like! These plugins cover the basic functionality of other popular extensions like plug3 and the Radiant Community Script (RCS).

  • Autowoot (Included)

    A timeless classic: woots every song automatically.
    installation instructions & source code

  • Chat Notifications (Included)

    Shows additional notification messages in chat when users join or leave, grab, meh or woot a song, and when the next song starts. Invaluable for room moderation!
    installation instructions & source code

  • Compact History (Included)

    ExtPlug plugin that shows a lot more history entries on one screen. Useful for Ctrl+F-ing!
    installation instructions & source code

  • Room Styles (Included)

    Make your room pop with a unique look and feel! Fully compatible with both the plug3 and RCS formats. Room owners don't have to do anything to support ExtPlug!
    installation instructions & source code

  • Meh Icons (Included)

    Shows meh icons in the user list, just like the standard woot and grab icons. Another invaluable moderation tool!
    installation instructions & source code

  • Show Deleted Messages (Included)

    Styles deleted chat messages instead of removing them. You guessed it, yet another invaluable moderation tool!
    installation instructions & source code

  • Hide Badges (Included)

    Hides user badges in chat, giving more breathing space to the actual messages.
    installation instructions & source code

Official Plugins

ExtPlug has a bunch of "official plugins". Some are included with ExtPlug by default, and others have to be installed manually.

  • User Profiles

    Integrates user profiles right into the plug.dj UI. Press a username anywhere to view their profile and play history, quickly and easily. Fantastic for identifying trolls!
    installation instructions & source code

  • Advanced Autocomplete

    Provides custom room-specific autocomplete options, especially for room bots. Includes a special preset for basicBot!
    installation instructions & source code

  • Chat Markup

    Applies simple Markdown/Slack/Reddit-like markup to chat messages.
    installation instructions & source code

  • Chat Scrollback

    Displays scrollback from previous sessions. An especially nice effect of that is that you won't lose your chat history after a refresh!
    installation instructions & source code

  • Message History

    Cycle through your previous messages using the up and down arrows in the chat box, similar to most IRC clients.
    installation instructions & source code

  • Rollover Blurb

    Fun little plugin that shows user blurb/bios in their hover popup!
    installation instructions & source code

  • Custom Emoji

    Provides custom room-specific emoji for use in the room chat. Room owners can add emoji images in the room settings.
    installation instructions & source code
